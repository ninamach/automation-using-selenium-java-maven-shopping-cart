## Environment:
- Java version: 17
- Maven version: 3.*
- Selenium HtmlUnitDriver: 2.52.0

- src/test/*

## Commands
- run: 
```bash
mvn clean package && java -jar target/selenium-java-shopping-cart-1.0.jar
```
- install: 
```bash
mvn clean install
```
- test: 
```bash
mvn clean test
```
